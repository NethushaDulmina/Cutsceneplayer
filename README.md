
# Cutscene Player

Simple CutScene Player for FiveM

## Preview

- ## [Youtube](https://youtu.be/46RWTUKTiRs)


## Installation

- Open Cutsceneplayer Zip
- Drag and Drop Into Server resoucres folder

```bash
start Cutsceneplayer
```
    
## Usage/Examples
- /cutscene [cut scene name]

```lua
/cutscene smun_intro
```
